# Como Clonar um repositório do GITHUB com o Visual Code
1. No GITHUB abra o repositório e clique no botão CODE -> Local -> copiar link do repositório
2. No Visual Code abra a pasta de seus projetos ( exemplo: Documentos/Python/ )
3. Abra o Terminal e digite o comando: git clone <cole aqui o link copiado> 
4. Tecle ENTER
5. No VS Code abra a pasta clonada 
6. No Terminal digite: git config user.name <digite aqui seu nome de usuário no Github>
7. Tecle ENTER
8. Ainda no Terminal digite: git config user.email <digite aqui seu email no Github>
